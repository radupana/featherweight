# Featherweight

[![codecov](https://codecov.io/gh/radupana/featherweight/branch/main/graph/badge.svg)](https://codecov.io/gh/radupana/featherweight)

Featherweight is a weightlifting companion designed for lifters who want comprehensive tracking and
program management without the complexity. Whether you're following a structured program or creating
your own workouts, Featherweight adapts to your training style.
